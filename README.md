# EdgeLabelTool

## Requirement
> `$ pip3 install numpy`  
`$ pip3 install opencv-python`  
make sure opencv version is at least 4.0.0

---
## Usage
>`python3 EdgeDetection.py --file [path/to/image/folder]`  
use trackbar below the window, clear edges remains
